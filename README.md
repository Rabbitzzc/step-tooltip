## ⭐️ Features

- 包体积更小，轻量无依赖
- 箭头一体化（伪类生成），可设置共同背景和border等


## 📦 Getting Started

### install
```sh
# npm 
npm install --save step-tooltip

# yarn
yarn add step-tooltip
```

### usage

```sh
import stepTooltip from 'step-tooltip'

stepTooltip()
```
