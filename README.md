## ⭐️ Features

- Lightweight, minify.
- Generate arrows from pseudo-classes(before/after).


🤚🎨🤚🎨demo: https://step-tooltip-example.logiczc.vercel.app/


## 📦 Getting Started

### install
```sh
# npm 
npm install --save step-tooltip

# yarn
yarn add step-tooltip
```

### usage

```sh
import stepTooltip from 'step-tooltip'

stepTooltip()
```
